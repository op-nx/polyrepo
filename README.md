# OpNx Polyrepo

> An open-source Nx plugin for polyrepos.

For [synthetic monorepos](https://nx.dev/docs/concepts/synthetic-monorepos), similar to [Nx Polygraph](https://nx.dev/docs/enterprise/polygraph).

No Nx Cloud or Nx Enterprise needed. Supports [self-hosted Nx cache](https://nx.dev/docs/guides/tasks--caching/self-hosted-caching).
