**Step 1: Add `isGitTag` to `git/detect.ts`**

Add an exported async function after the existing `getCurrentRef`:

```typescript
export async function isGitTag(cwd: string, ref: string | undefined): Promise<boolean> {
  if (!ref) {
    return false;
  }

  try {
    await execGitOutput(['show-ref', '--verify', `refs/tags/${ref}`], cwd);

    return true;
  } catch {
    return false;
  }
}
```

This uses `git show-ref --verify refs/tags/<ref>` which exits 0 if the tag exists and non-zero if not. Works for any tag name without assumptions about naming conventions.
