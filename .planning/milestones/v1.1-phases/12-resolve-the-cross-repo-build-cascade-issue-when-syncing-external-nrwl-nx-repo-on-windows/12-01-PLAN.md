<objective>
Isolate host targetDefaults from external proxy targets and add environment isolation to the proxy executor.

Purpose: Fix the root cause of the amplified cross-repo build cascade. Host `targetDefaults` (e.g., `test.dependsOn: ["^build"]`) currently leak into all ~150 external project proxy targets because `dependsOn` is `undefined` on proxy targets, and Nx treats `undefined` as "apply targetDefaults." Additionally, the proxy executor needs SQLite and daemon isolation via environment variables so child Nx processes do not conflict with the host.

Output: Updated `transform.ts` with `rewriteDependsOn` function, updated `executor.ts` with env var isolation, and comprehensive tests for both.
</objective>

    ```typescript
    function rewriteDependsOn(
      rawDependsOn: unknown,
      repoAlias: string,
    ): TargetConfiguration['dependsOn'] {
      if (!Array.isArray(rawDependsOn)) {
        return [];
      }

      return rawDependsOn.map((entry: unknown) => {
        if (typeof entry === 'string') {
          return entry;
        }

        if (isRecord(entry) && typeof entry['target'] === 'string') {
          const result: Record<string, unknown> = { ...entry };

          if (Array.isArray(entry['projects'])) {
            result['projects'] = (entry['projects'] as unknown[]).map((p: unknown) =>
              typeof p === 'string' && !p.startsWith('tag:')
                ? `${repoAlias}/${p}`
                : p
            );
          }

          return result;
        }

        return entry;
      });
    }
    ```
